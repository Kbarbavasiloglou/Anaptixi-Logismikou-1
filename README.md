# ΣΥΝΤΟΜΗ ΠΕΡΙΓΡΑΦΗ ΣΕΝΑΡΙΟΥ
Η παρούσα εφαρμογή υλοποιεί το use case "Πραγματοποίηση Δωρεάς

Ο απλός χρήστης του συστήματος (Donor) μπορεί να προχωρήσει στις παρακάτω ενέργειες:

-εισαγωγή των στοιχείων του
-επιλογή ποσού δωρεάς
-δήλωση τρόπου πληρωμής
-καταχώρηση ημερομηνίας δωρεάς

κι ανάλογα λαμβάνει μήνυμα επιτυχίας και απόδειξης πληρωμής ή αποτυχίας

Ο διαχειριστής (Admin user) μπορεί να συνδεθεί στο σύστημα και να δει αναφορά με όλες
τις καταγεγραμμένες δωρεές.
